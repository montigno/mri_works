Prerequisite

mri_works is developped and tested under Python 3.6 and under Ubuntu 20.04, so it is recommended to use Python 3.6.
Python 3.8 or more is not (for the moment) recommended, Python 3.7 should work.
mri_works is also tested under Windows 10 which works well overall, however the FSL module will not work.

1 - Install Python 3.6, pip and virtualenv

- download Python 3.6.8 for Windows 10 64-bits here
- execute the downloaded file.
- locate the directory where Python is installed, often in C:\Users\your_user_name\AppData\Local\Programs\Python\Python3.6
- add Python to the windows path (explanation here)
- download the get-pip.py file here
- open the Command Prompt (type “cmd” into the search bar and click on “Command Prompt”)
- go to the directory where get-pip.py file is located (e.g. C:\Users\your_user_name\Downloads\) 
cd C:\Users\your_user_name\Downloads\
- type : 
python get-pip.py
- pip will be installed (normaly) in C:\Users\your_user_name\AppData\Local\Programs\Python\Python3.6\Scripts
- add pip to the windows path (explanation here) and retart the Command Prompt.

2 - Download mri_works

- download the release of mri_works from the github homepage to the 'Download' repertory
- decompress the downloaded file
- go to the unzipped directory where the setup.py file is located 
cd %homepath%\Downloads\mri_works-master
(go to the unzipped directory)
